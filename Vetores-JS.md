# Trabalhando com Vetores

### Exemplo 1
```
let pokemon = ["Pikachu", "Charmander", "Bulbasaur"] //vetor

pokemon.pop() //remove o que está armazenado na posição 0
pokemon.shift() //remove o que está armazenado na última posição

console.log(pokemon)

console.log(pokemon[0])
console.log(pokemon[1])
console.log(pokemon[2])
```
